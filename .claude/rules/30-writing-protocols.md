# Writing protocol discipline

When the user requests writing assistance, load the appropriate protocol from `skills-library/`:

| Genre | Protocol file |
|-------|--------------|
| Scientific paper | `skills-library/paper-writing-protocol.md` |
| Newspaper op-ed / guest essay | `skills-library/oped-writing-protocol.md` |
| Long-form ideas essay (Atlantic-style) | `skills-library/atlantic-ideas-writing-protocol.md` |
| Pitch for ideas essay | `skills-library/altac-writing-pitch-protocol.md` |

## Rules

- Load the protocol before drafting. Do not write from general knowledge alone.
- Follow the protocol's structure, voice, and revision guidance.
- Preserve the user's academic voice and expertise — translate, don't flatten.
- If the genre is ambiguous, ask before drafting.
